Bienvenido a la sección de control de flujo. En esta sección aprenderás a controlar el flujo de ejecución de tus programas en Rust. Esto te permitirá tomar decisiones y repetir tareas de forma automática. Tambien veremos cómo trabajar con funciones en Rust.

## Temas a tratar

- Condicionales: Veremos cómo tomar decisiones en nuestros programas utilizando condicionales.

- Uso de match: Aprenderemos a utilizar `match` para tomar decisiones en Rust. `match` es una expresión muy poderosa que nos permite comparar un valor con una serie de patrones y ejecutar código dependiendo del patrón que coincida.

- Bucles: Aprenderemos a repetir tareas de forma automática utilizando bucles. Usaremos diferentes tipos de bucles que nos ofrece Rust.

- Funciones: Veremos cómo definir y utilizar funciones en Rust. También veremos cómo devolver valores desde una función.

## Objetivos de aprendizaje

- Aprender a utilizar condicionales para tomar decisiones en Rust.

- Aprender a utilizar `match` para tomar decisiones en Rust.

- Aprender a utilizar bucles para repetir tareas de forma automática.

- Aprender a definir y utilizar funciones en Rust.

¡Vamos a empezar! 🚀
