El bloque `if-else` en Rust es una estructura de control de flujo que permite que el programa tome decisiones basadas en una condición booleana. Si la condición es `true`, se ejecuta un bloque de código; si es `false`, se puede ejecutar una alternativa especificada por `else`. Esta estructura permite que el programa realice diferentes acciones dependiendo de la evaluación de condiciones.

Imagina que estás decidiendo si llevar un paraguas antes de salir de casa. Si ves que el cielo está nublado, decides llevar el paraguas. Si está despejado, decides no llevarlo. Aquí, "ver si está nublado" es la condición, y las dos acciones posibles (llevar o no el paraguas) dependen del resultado de esa evaluación, similar a cómo un `if-else` evalúa una condición y toma diferentes acciones en un programa.
