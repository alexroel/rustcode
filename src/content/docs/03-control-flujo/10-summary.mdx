En esta sección, hemos aprendido sobre el control de flujo en Rust, que nos permite tomar decisiones y ejecutar instrucciones de manera condicional. Aquí hay un resumen de los conceptos clave que hemos cubierto:

## Temas aprendidos

- **Condicionales:** Utilizamos la declaración `if` para ejecutar un bloque de código si se cumple una condición específica. También podemos usar `else` y `else if` para manejar múltiples condiciones.

- **Coincidencia de patrones:** Utilizamos la declaración `match` para comparar un valor con una serie de patrones y ejecutar el código correspondiente al patrón coincidente.

- **Bucles:** Aprendimos sobre los bucles `loop`, `while` y `for` en Rust, que nos permiten ejecutar un bloque de código repetidamente hasta que se cumpla una condición específica.

- **Funciones:** Definimos funciones en Rust utilizando la palabra clave `fn` y pasamos parámetros a las funciones para personalizar su comportamiento.

- **Expresiones:** Aprendimos que las expresiones en Rust devuelven un valor y pueden ser utilizadas en diferentes contextos, como asignaciones y declaraciones de retorno.

- **Refactorización de nuesto juego:** Refactorizamos nuestro juego de adivinar el número para hacerlo más modular y fácil de entender. También agregamos mejoras, como la capacidad de elegir niveles de dificultad.

## Ahora eres capaz de ...

- Utilizar condicionales para tomar decisiones en tu código.
- Utilizar la declaración `match` para comparar valores con patrones.
- Utilizar bucles para ejecutar un bloque de código repetidamente.
- Definir y utilizar funciones en Rust.
- Utilizar expresiones para devolver valores y realizar tareas repetitivas.
