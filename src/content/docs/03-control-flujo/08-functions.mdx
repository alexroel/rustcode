Las funciones son bloques de código que realizan una tarea específica. En Rust, las funciones se definen con la palabra clave `fn`, seguida del nombre de la función y una lista de parámetros entre paréntesis. A continuación, se muestra un ejemplo de una función simple que imprime un mensaje en la consola:

```rust
fn saludar() {
    println!("Hola desde la función saludar!");
}
```

Para llamar a una función, simplemente escriba su nombre seguido de paréntesis. Por ejemplo:

```rust

fn saludar() {
    hola();
}
```

En este caso, la función `main` llama a la función `saludar`, que imprime el mensaje "Hola desde la función saludar!" en la consola.

## Funciones con parámetros

Las funciones pueden aceptar parámetros que se utilizan para pasar información a la función. Los parámetros se definen entre paréntesis después del nombre de la función. A continuación, se muestra un ejemplo de una función que acepta un parámetro de tipo `String`:

```rust

fn saludar(nombre: String) {
    println!("¡Hola, {}!", nombre);
}
```

Para llamar a esta función, simplemente pase un argumento de tipo `String` entre paréntesis. Por ejemplo:

```rust
fn main() {
    saludar(String::from("Mundo"));
}
```

En este caso, la función `main` llama a la función `saludar` con el argumento `String::from("Mundo")`, que imprime el mensaje "¡Hola, Mundo!" en la consola.

## Funciones con valor de retorno

Las funciones pueden devolver un valor utilizando la palabra clave `return` seguida del valor que se desea devolver. A continuación, se muestra un ejemplo de una función que devuelve un valor de tipo `String`:

```rust
fn saludar(nombre: String) -> String {
    return format!("¡Hola, {}!", nombre);
}
```

Para llamar a esta función y utilizar su valor de retorno, simplemente asigne el resultado a una variable. Por ejemplo:

```rust
fn main() {
    let mensaje = saludar(String::from("Mundo"));
    println!("{}", mensaje);
}
```

En este caso, la función `main` llama a la función `saludar` con el argumento `String::from("Mundo")` y asigna el resultado a la variable `mensaje`, que luego imprime el mensaje "¡Hola, Mundo!" en la consola.

## Sin usar `return`

En Rust, la última expresión de una función se considera su valor de retorno. Por lo tanto, no es necesario utilizar la palabra clave `return` explícitamente si se desea devolver el valor de la última expresión. A continuación, se muestra un ejemplo de una función que devuelve un valor de tipo `String` sin utilizar `return`:

```rust
fn saludar(nombre: String) -> String {
    format!("¡Hola, {}!", nombre)
}
```

En este caso, la función `saludar` devuelve el resultado de la llamada a `format!("¡Hola, {}!", nombre)`, que es la última expresión de la función.

## Funciones con parámetros con nombre

En Rust, los parámetros de una función pueden tener nombres asociados que se utilizan para identificar los valores pasados a la función. A continuación, se muestra un ejemplo de una función que acepta un parámetro con nombre:

```rust

fn saludar(nombre: String) {
    println!("¡Hola, {}!", nombre);
}
```

Para llamar a esta función con un parámetro con nombre, simplemente especifique el nombre del parámetro seguido de dos puntos y el valor que desea pasar. Por ejemplo:

```rust

fn main() {
    saludar(nombre: String::from("Mundo"));
}
```


## Funciones con múltiples parámetros

En este ejemplo vamos a crear una función que recibe dos parámetros de tipo `i32` y devuelve la suma de ambos:

```rust
fn sumar(a: i32, b: i32) -> i32 {
    a + b
}
```

Para llamar a esta función, simplemente pase dos argumentos de tipo `i32` entre paréntesis. Por ejemplo:

```rust
fn main() {
    let resultado = sumar(5, 3);
    println!("El resultado de la suma es: {}", resultado);
}
```

En este caso, la función `main` llama a la función `sumar` con los argumentos `5` y `3`, y asigna el resultado a la variable `resultado`, que luego imprime el mensaje "El resultado de la suma es: 8" en la consola.

## Funciones con múltiples valores de retorno

En Rust, las funciones pueden devolver múltiples valores utilizando tuplas. A continuación, se muestra un ejemplo de una función que devuelve dos valores de tipo `i32`:

```rust

fn dividir(a: i32, b: i32) -> (i32, i32) {
    (a / b, a % b)
}
```

Para llamar a esta función y utilizar sus valores de retorno, simplemente asigne el resultado a una variable de tipo tupla. Por ejemplo:

```rust

fn main() {
    let (cociente, resto) = dividir(10, 3);
    println!("El cociente es: {}, el resto es: {}", cociente, resto);
}
```

En este caso, la función `main` llama a la función `dividir` con los argumentos `10` y `3`, y asigna los valores de retorno a las variables `cociente` y `resto`, que luego imprime los mensajes "El cociente es: 3, el resto es: 1" en la consola.

## Conclusión

En este capítulo, aprendimos a definir y utilizar funciones en Rust. Las funciones son bloques de código que realizan una tarea específica y pueden aceptar parámetros, devolver valores y tener múltiples valores de retorno. Las funciones son una parte fundamental de la programación en Rust y se utilizan para organizar y reutilizar el código de manera eficiente.
