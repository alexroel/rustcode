En esta sección, aprendiste los conceptos básicos de Rust, cómo instalarlo y configurar tu entorno de desarrollo, y cómo escribir y compilar tu primer programa en Rust. Aquí hay un resumen de lo que cubrimos:

- **Introducción a Rust**: Descubriste qué es Rust, sus características clave y por qué es un lenguaje de programación moderno y seguro.

- **Instalación de Rust**: Aprendiste cómo instalar Rust en tu sistema utilizando `rustup`, el instalador oficial de Rust.

- **Configuración de VSCode para Rust**: Configuraste Visual Studio Code para programar en Rust, instalando la extensión `rust-analyzer` y configurando el entorno de desarrollo.

- **Creación de un "Hola Mundo" en Rust**: Creaste y ejecutaste un simple programa "Hola Mundo" en Rust, aprendiendo sobre la estructura básica de un programa Rust y cómo compilarlo.

## ¿Qué ya sabes?
