Bienvenido a la sección de genéricos. En esta sección se explican los conceptos básicos de los genéricos en Rust. Se explica cómo se definen y cómo se utilizan. 

Los genéricos son una característica muy poderosa de Rust que permite escribir código que es independiente del tipo de dato. Esto permite reutilizar código y escribir funciones y estructuras de datos que pueden trabajar con diferentes tipos de datos.

En esta sección se explican los conceptos básicos de los genéricos en Rust. Se explica cómo se definen y cómo se utilizan.

## Temas a tratar

- **Introducción a los genéricos**: Se explica qué son los genéricos y por qué son útiles.
- **Funciones genéricas**: Se explica cómo se definen funciones genéricas. Se muestra cómo se pueden escribir funciones que trabajan con diferentes tipos de datos.
- **Estructuras genéricas**: Se explica cómo se definen estructuras genéricas. Se muestra cómo se pueden escribir estructuras de datos que trabajan con diferentes tipos de datos.
- **Enums genéricos**: Se explica cómo se definen enums genéricos. Se muestra cómo se pueden escribir enums que trabajan con diferentes tipos de datos.
- **Traits genéricos**: Se explica cómo se definen traits genéricos. Se muestra cómo se pueden escribir traits que trabajan con diferentes tipos de datos.


## Objetivos

El objetivo de esta sección es aprender a utilizar los genéricos en Rust. Al final de esta sección, deberías ser capaz de:

- Entender qué son los genéricos y por qué son útiles.
- Definir funciones genéricas.
- Definir estructuras genéricas.
- Definir enums genéricos.
- Definir traits genéricos.
