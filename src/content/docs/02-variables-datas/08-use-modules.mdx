En Rust, puedes usar módulos externos para importar código de otras bibliotecas o archivos. Esto te permite reutilizar código y mantener tus proyectos organizados. Los módulos en Rust también se llaman *crates*. Un *crate* es una colección de módulos que pueden ser compilados y compartidos. 

## Importar un Crate

Para agregar un *crate* externo a tu proyecto, puedes usar la palabra clave `cargo add` seguida del nombre del *crate*. Por ejemplo, para importar el *crate* `rand`, puedes agregar la siguiente línea en la terminal:

```bash
cargo add rand
```

Esto agregará el *crate* `rand` a tu archivo `Cargo.toml` y descargará el *crate* en tu proyecto.

Tambien puedes agregar el *crate* manualmente en tu archivo `Cargo.toml`:

```toml
[dependencies]
rand = "0.9.0"
```

## Usar un Crate

Para usar un *crate* en tu proyecto, puedes agregar la siguiente línea al principio de tu archivo:

```diff lang="rs"
use std::io::{stdin, stdout, Write};

+// Importar el crate rand
+use rand::random_range;


fn main() {
    println!("Mi primer juego con Rust");

+    // Generar un número aleatorio entre 1 y 100
+    let secret_number = random_range(1..101);
+    println!("El número secreto es: {}", secret_number);    

    let mut user_choese = String::new();

    print!("Ingres un número entre (0 y 100):");
    // Vacia el bufer
    stdout().flush().expect("No se pudo vaciar stdout"); 

    //Ingresa los datos desde teclado
    stdin().read_line(&mut user_choese).expect("Error al leer la línea");

    // Convertir de String a Integer
    let user_choese:i32 = user_choese.trim().parse().expect("¡Ingrese un número por favor!");
    
    println!("El número ingresado por el usaurio es: {}", user_choese);
   
}

```

- **`use rand::random_range`**: Importa la función `random_range` del *crate* `rand`. Esta función se utiliza para generar un número aleatorio dentro de un rango específico.

- **`random_range(1..101)`**: Genera un número aleatorio entre 1 y 100.

Ahora puedes usar el *crate* `rand` en tu proyecto para generar números aleatorios. Los *crates* externos te permiten agregar funcionalidades adicionales a tu proyecto y reutilizar código de otras bibliotecas.

## Conclusiones

En Rust, puedes usar *crates* externos para importar código de otras bibliotecas o archivos. Los *crates* te permiten reutilizar código y mantener tus proyectos organizados. Puedes agregar un *crate* a tu proyecto usando la palabra clave `cargo add` o manualmente en tu archivo `Cargo.toml`. Luego, puedes usar las funciones y estructuras del *crate* en tu proyecto importándolas al principio de tu archivo.
