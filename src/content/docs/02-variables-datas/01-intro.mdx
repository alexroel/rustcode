Bienvenido a la sección de variables y tipos de datos en Rust. En esta sección aprenderás sobre las variables y los tipos de datos en Rust. Empezaremos con la estructura básica de un proyecto y luego aprenderemos cómo declarar variables y los tipos de datos en Rust.

import { Steps } from '@astrojs/starlight/components';

## Temas a tratar en esta sección

<Steps>

1. **Estructura básica de un proyecto**: Aprenderemos sobre la estructura básica de un proyecto en Rust.

2. **Variables**: Aprenderemos cómo declarar variables en Rust.

3. **Tipos de datos**: Aprenderemos sobre los tipos de datos en Rust.

4. **Datos de entrada**: Aprenderemos cómo obtener datos de entrada en Rust.

5. **Modulos externos**: Aprenderemos cómo importar modulos externos en Rust.

6. **Operadores**: Aprenderemos sobre los operadores en Rust.

</Steps>

## Objetivos de esta sección

Al final de esta sección, serás capaz de:

- Entender la estructura básica de un proyecto en Rust.
- Declarar variables en Rust.
- Conocer los tipos de datos en Rust.
- Obtener datos de entrada en Rust.
- Importar modulos externos en Rust.
- Entender los operadores en Rust.

¡Empecemos! 🚀
