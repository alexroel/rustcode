Para empezar con un proyecto en Rust, el comando `cargo` es fundamental. Cargo es el gestor de paquetes y compilación en Rust. A continuación, creamos un nuevo proyecto llamado "game-gues" con el siguiente comando:

```bash
cargo new game-gues
```

Este comando crea una estructura de directorios y archivos básicos para tu proyecto. Dentro de la carpeta creada, encontrarás un archivo principal llamado `main.rs`, que contendrá la función principal del programa.

## La función principal de Rust

En Rust, todo programa debe tener una función principal que actúe como punto de entrada. A continuación, mostramos cómo se estructura esta función:

```rust
// Función principal del programa
fn main() {
    /*
    - Después de una sangría se escribe el código.
    - Cada instrucción termina en punto y coma.
    - Si no termina en punto y coma, Rust entiende que la instrucción continúa.
    */
}
```

## Sangría del código

La sangría es una convención clave en Rust. Cada bloque de código que pertenece a una función o estructura debe estar correctamente indentado para mejorar la legibilidad. En este ejemplo, el bloque de código dentro de la función `main` está indentado, y cada instrucción dentro de la función debe terminar con un punto y coma (`;`), a menos que se trate de una expresión que no devuelva un valor.

## Comentarios en Rust

Los comentarios en Rust se pueden escribir de dos formas: de una sola línea o de varias líneas. A continuación, se muestra cómo se escriben los comentarios en Rust:

```rust
// Este es un comentario de una sola línea

/*
Este es un comentario de varias líneas.
Puede abarcar varias líneas de código.
*/
```

## ¿Qué es un macro en Rust?

Un macro en Rust es una herramienta que permite generar código en tiempo de compilación. Los macros permiten escribir código más flexible y eficiente. Uno de los macros más utilizados en Rust es `println!`.

**Macro `println!`**

El macro `println!` se utiliza para imprimir texto en la consola. La sintaxis es la siguiente:

```rust
fn main() {
    println!("Mi primer juego con Rust");
}
```

Este simple programa imprime el texto `"Mi primer juego con Rust"` en la consola cuando se ejecuta. El signo de exclamación (`!`) después de `println` indica que es un macro, no una función normal.

Con esta breve introducción, hemos cubierto la estructura básica de un programa en Rust, la importancia de la función `main`, la correcta sangría del código y el uso de macros como `println!`. Estos son los primeros pasos para comenzar a escribir programas en Rust, y te proporcionan una base sólida para avanzar en el aprendizaje de este poderoso lenguaje.
