Las cadenas de texto en Rust son representadas por el tipo de dato `String`. En este capítulo, aprenderás cómo declarar y trabajar con cadenas de texto en Rust. 

Las cadenas de texto en Rust son inmutables y no pueden ser modificadas una vez que son creadas. Para crear una cadena de texto `String`, puedes usar el método `to_string()`.

```rust
let mut name = "Alex".to_string();
name.push_str(" en el mundo de Rust");

println!("{}", name); // Alex en el mundo de Rust
```

En el ejemplo anterior, hemos creado una cadena de texto `String` llamada `name` y hemos añadido la cadena de texto `" in Wonderland"` al final de la cadena de texto `name` utilizando el método `push_str()`.

## Trabajando con cadenas de texto

En Rust, puedes trabajar con cadenas de texto utilizando el tipo compuesto `String`. A continuación, se presentan algunas operaciones comunes que puedes realizar con cadenas de texto en Rust:

**1. Creación de una cadena de texto `String`**:

```rust
let message = String::new();
message.push_str("¡Hola, mundo!");

println!("{}", message); // ¡Hola, mundo!
```

En este ejemplo, hemos creado una cadena de texto `String` vacía llamada `message` usando el método `::new()` y hemos añadido la cadena de texto `"¡Hola, mundo!"` al final de la cadena de texto `message` utilizando el método `push_str()`.

**2. Creando una cadena de texto inicializada**:

```rust
let message = String::from("¡Hola, mundo!");
println!("{}", message); // ¡Hola, mundo!
```

En este ejemplo, hemos creado una cadena de texto `String` llamada `message` inicializada con el valor `"¡Hola, mundo!"` utilizando el método `::from()`.

**3. Concatenación de cadenas de texto**:

```rust
let mut message = String::from("¡Hola, ");
message.push_str("mundo!");

println!("{}", message); // ¡Hola, mundo!
```

En este ejemplo, hemos creado una cadena de texto `String` llamada `message` con el valor `"¡Hola, "` y hemos añadido la cadena de texto `"mundo!"` al final de la cadena de texto `message` utilizando el método `push_str()`.

**4. Concatenación de cadenas de texto con con formato**:

```rust 
let name = "Alex";

let message = format!("¡Hola, {}!", name);
println!("{}", message); // ¡Hola, Alex!
```

En este ejemplo, hemos creado una cadena de texto `String` llamada `message` utilizando el método `format!()` para concatenar la cadena de texto `"¡Hola, "` con el valor de la variable `name` y la cadena de texto `"!"`.


## Conclusiones

En este capítulo, hemos aprendido cómo trabajar con cadenas de texto en Rust utilizando el tipo de dato `String`. Las cadenas de texto en Rust son inmutables y no pueden ser modificadas una vez que son creadas. Puedes utilizar métodos como `push_str()` y `format!()` para trabajar con cadenas de texto en Rust.
