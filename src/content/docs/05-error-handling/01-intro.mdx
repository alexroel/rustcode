Bienvenido a la sección de manejo de errores en Rust. En esta sección, aprenderemos sobre cómo Rust maneja los errores y cómo podemos utilizar el sistema de tipos del lenguaje para prevenir errores en tiempo de compilación. Aquí hay un resumen de los temas que cubriremos:

## Temas a tratar
En esta sección, cubriremos los siguientes temas:

- **Errores en Rust:** Aprenderemos sobre los diferentes tipos de errores en Rust y cómo se manejan en el lenguaje.

- **Uso de tipo Option:** Utilizaremos el tipo `Option` para representar valores opcionales y manejar casos en los que un valor puede ser nulo.

- **Uso de tipo Result:** Utilizaremos el tipo `Result` para representar operaciones que pueden fallar y manejar errores de manera segura.

- **Manejo de errores con `match`:** Aprenderemos a utilizar la declaración `match` para manejar errores y tomar decisiones basadas en el resultado de una operación.

- **Manejo de errores con `unwrap`:** Aprenderemos a utilizar el método `unwrap` para desempaquetar valores de tipo `Option` y `Result` de manera segura.

- **Manejo de errores con `expect`:** Aprenderemos a utilizar el método `expect` para manejar errores de manera más descriptiva y proporcionar mensajes de error personalizados.

- **Manejo de errores con `?`:** Aprenderemos a utilizar el operador `?` para propagar errores de manera eficiente y concisa.


## Objetivos de aprendizaje
Al final de esta sección, serás capaz de:

- Comprender los diferentes tipos de errores en Rust y cómo se manejan en el lenguaje.

- Utilizar los tipos `Option` y `Result` para manejar valores opcionales y errores de manera segura.

- Utilizar la declaración `match` para manejar errores y tomar decisiones basadas en el resultado de una operación.

- Utilizar los métodos `unwrap`, `expect` y el operador `?` para manejar errores de manera eficiente y concisa.
