En esta lección, aprenderás cómo usar arrays y tuplas en Rust. Aprenderás cómo definir arrays y tuplas, cómo acceder a los elementos de un array o tupla y cómo usar arrays y tuplas en tu código. Los arrays y tuplas son tipos de datos compuestos que te permiten almacenar varios valores juntos. Puedes usar arrays para almacenar una colección de elementos del mismo tipo y tuplas para almacenar una colección de elementos de diferentes tipos.

## Definición de arrays

Los arrays en Rust se definen usando la palabra clave `let` seguida del nombre del array, dos puntos `:` y el tipo de dato del array, seguido de corchetes `[]` con los elementos del array separados por comas. Aquí tienes un ejemplo de cómo definir un array en Rust:

```rust
let numeros = [1, 2, 3, 4, 5];
```

En este ejemplo, hemos definido un array llamado `numeros` con cinco elementos: `1`, `2`, `3`, `4` y `5`. El tipo de dato del array es `[i32; 5]`, que significa que el array contiene cinco elementos de tipo `i32` (entero de 32 bits).

## Acceso a los elementos de un array

Puedes acceder a los elementos de un array usando la notación de corchetes `[]` seguida del índice del elemento que quieres acceder. Los índices de los elementos de un array comienzan en `0` y van hasta `n-1`, donde `n` es el número de elementos del array. Aquí tienes un ejemplo de cómo acceder a los elementos del array `numeros`:

```rust
println!("Primer número: {}", numeros[0]);
println!("Segundo número: {}", numeros[1]);
``` 

En este ejemplo, estamos accediendo al primer y segundo elemento del array `numeros` y almacenando sus valores en las variables `primer_numero` y `segundo_numero`.

## Definición de tuplas

Las tuplas en Rust se definen usando la palabra clave `let` seguida del nombre de la tupla, dos puntos `:` y una lista de tipos de datos separados por comas entre paréntesis `()`. Aquí tienes un ejemplo de cómo definir una tupla en Rust:

```rust
let persona = ("Alex", 30, 'M');
```

En este ejemplo, hemos definido una tupla llamada `persona` con tres elementos: `"Alex"` de tipo `&str`, `30` de tipo `i32` y `'M'` de tipo `char`. Las tuplas te permiten combinar varios valores de diferentes tipos en una sola variable.

## Acceso a los elementos de una tupla

Puedes acceder a los elementos de una tupla usando la notación de punto `.` seguida del índice del elemento que quieres acceder. Los índices de los elementos de una tupla comienzan en `0` y van hasta `n-1`, donde `n` es el número de elementos de la tupla. Aquí tienes un ejemplo de cómo acceder a los elementos de la tupla `persona`:

```rust
println!("Nombre: {}", persona.0);
println!("Edad: {}", persona.1);
println!("Género: {}", persona.2);
```

En este ejemplo, estamos accediendo a los elementos de la tupla `persona` y mostrando sus valores en la consola. Hemos usado los índices `0`, `1` y `2` para acceder al nombre, edad y género de la persona, respectivamente.

## Podemos crear un array para estructuras

Podemos crear un array de estructuras en Rust. Aquí tienes un ejemplo de cómo hacerlo:

```rust
struct Estudiante {
    nombre: String,
    edad: u8,
}

impl Estudiante {
    fn nuevo(nombre: &str, edad: u8) -> Self {
        Self {
            nombre: String::from(nombre),
            edad,
        }
    }
}

fn main() {
    let estudiantes = [
        Estudiante::nuevo("Alex", 30),
        Estudiante::nuevo("Eva", 25),
        Estudiante::nuevo("Juan", 28),
    ];

    for estudiante in &estudiantes {
        println!("Nombre: {}", estudiante.nombre);
        println!("Edad: {}", estudiante.edad);
    }
}
```

En este ejemplo, hemos definido una estructura `Estudiante` con dos campos: `nombre` de tipo `String` y `edad` de tipo `u8`. Hemos definido un método asociado `nuevo` en la estructura `Estudiante` para crear instancias de la estructura con los valores proporcionados. Hemos creado un array `estudiantes` de tres elementos de tipo `Estudiante` y hemos inicializado cada elemento con un estudiante diferente. Hemos recorrido el array `estudiantes` con un bucle `for` y hemos mostrado el nombre y la edad de cada estudiante en la consola.

## Estructura como tupla

Podemos usar una estructura como una tupla en Rust. Aquí tienes un ejemplo de cómo hacerlo:

```rust
strcut Grados {char, char, char, i32}

fn main() {
    let grados = Grados('A', 'B', 'C', 100);
    println!("Grados: {}, {}, {}, {}", grados.0, grados.1, grados.2, grados.3);
}
```

En este ejemplo, hemos definido una estructura `Grados` con cuatro campos de tipo `char` y `i32`. Hemos creado una instancia de la estructura `Grados` con los valores `'A'`, `'B'`, `'C'` y `100`. Hemos accedido a los campos de la estructura `Grados` usando la notación de punto `.` y mostrado sus valores en la consola.
