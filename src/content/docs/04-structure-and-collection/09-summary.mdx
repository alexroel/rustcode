En esta sección, aprendiste cómo estructurar y colecciones de datos en Rust. Aprendiste cómo definir estructuras, cómo crear instancias de estructuras, cómo acceder a los campos de una estructura, cómo implementar métodos para estructuras, cómo definir enumeraciones, cómo definir arrays y tuplas, cómo definir vectores, cómo definir mapas hash y cómo usar estructuras y colecciones de datos en tu código.
