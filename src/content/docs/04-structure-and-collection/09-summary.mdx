En esta sección, aprendiste cómo estructurar y colecciones de datos en Rust. Aprendiste cómo definir estructuras, cómo crear instancias de estructuras, cómo acceder a los campos de una estructura, cómo implementar métodos para estructuras, cómo definir enumeraciones, cómo definir arrays y tuplas, cómo definir vectores, cómo definir mapas hash y cómo usar estructuras y colecciones de datos en tu código.

## Resumen de conceptos

- **Estructuras**: Las estructuras son tipos de datos personalizados que te permiten combinar múltiples valores en una sola unidad. Puedes definir una estructura usando la palabra clave `struct` seguida de un nombre y una lista de campos.

- **Instancias de estructuras**: Puedes crear instancias de una estructura usando la palabra clave `struct` seguida de un nombre y una lista de valores para los campos de la estructura.

- **Acceso a los campos de una estructura**: Puedes acceder a los campos de una estructura usando la notación de punto `.` seguida del nombre del campo.

- **Métodos para estructuras**: Puedes implementar métodos para estructuras usando la palabra clave `impl` seguida del nombre de la estructura y el bloque de código que define el método.

- **Enumeraciones**: Las enumeraciones son tipos de datos que te permiten definir un conjunto finito de valores posibles. Puedes definir una enumeración usando la palabra clave `enum` seguida de un nombre y una lista de variantes.

- **Arrays y tuplas**: Los arrays y las tuplas son colecciones de valores fijos en Rust. Los arrays tienen un tamaño fijo y todos los elementos deben ser del mismo tipo, mientras que las tuplas pueden tener diferentes tipos de elementos y una longitud variable.

- **Vectores**: Los vectores son colecciones de valores dinámicos en Rust. Puedes agregar y quitar elementos de un vector según sea necesario.

- **Mapas hash**: Los mapas hash son colecciones de elementos clave-valor en Rust. Puedes usar mapas hash para almacenar pares de valores juntos y acceder a ellos por clave.
