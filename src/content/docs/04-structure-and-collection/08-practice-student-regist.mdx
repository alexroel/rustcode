En esta práctica, aprenderás cómo usar estructuras y colecciones de datos en Rust para crear un registro de estudiantes. Aprenderás cómo definir una estructura para representar a un estudiante, cómo crear un vector de estudiantes, cómo agregar y quitar estudiantes del registro y cómo buscar estudiantes por nombre.

## Requerimientos

Para completar esta práctica, necesitamos tener encuenta los siguientes requerimientos:

- El programa tiene que tener un menú con las siguientes opciones:
  1. Agregar estudiante
  2. Eliminar estudiante
  3. Listar estudiantes
  4. Salir

- Para agregar un estudiante, el programa debe solicitar el nombre y la edad del estudiante al usuario y agregarlo al registro de estudiantes.

- Para eliminar un estudiante, el programa debe solicitar el número de registro del estudiante a eliminar y eliminarlo del registro de estudiantes.

- Para listar los estudiantes, el programa debe mostrar una lista de todos los estudiantes en el registro.

- El programa debe salir cuando el usuario seleccione la opción de salir.

## Solución de la práctica

Para completar esta práctica, sigue los siguientes pasos:

1. Define una estructura llamada `Estudiante` con los campos `nombre` de tipo `String` y `edad` de tipo `u8`.
2. Crea un vector llamado `registro` para almacenar los estudiantes.
3. Implementa un bucle infinito que muestre el menú y solicite al usuario que seleccione una opción.
4. Implementa la lógica para cada opción del menú:
   - Para la opción de agregar estudiante, solicita el nombre y la edad del estudiante al usuario y agrega un nuevo estudiante al registro.
   - Para la opción de eliminar estudiante, solicita el número de registro del estudiante a eliminar y elimina el estudiante del registro.
   - Para la opción de listar estudiantes, muestra una lista de todos los estudiantes en el registro.
   - Para la opción de salir, finaliza el programa.
